Class ShopVendorComponent

A vending machine that sells items for a currency controlled by events. Does not need restocking. Another component must handle Content.Shared.DeltaV.VendingMachines.ShopVendorBalanceEvent and Content.Shared.DeltaV.VendingMachines.ShopVendorPurchaseEvent to work.
