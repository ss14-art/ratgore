ChatMessageAction

Makes the target or the user to send a chat message.

Messages are locale-based, their keys follow the form of "interaction-[verb id]-[message loc prefix]-[index]". The index parameter is a random integer from 1 to NumMessages.

Similarly to interaction verb locales, {$user}, {$target} amd {$used} arguments are passed to the locales retrieved by this action.

A special proxy action that swaps the target and the user for the proxied action. This effectively means that in most cases the proxied action will be applied to the user even if it's meant for target.
